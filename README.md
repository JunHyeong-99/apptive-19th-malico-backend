# Marico

## 프로젝트 소개

패션에 관심 있지만 스타일링은 미흡한 20대 남성에게 
스타일리스트 매칭을 통한 개인 맞춤형 스타일링 제공

## 개발 기간
2023.09 ~ 2024.01

## 기술 스택

### 개발 환경
- JDK17
- Gradle
- Springboot 3.0.5
- Spring Data JPA

### 보안
- SpringSecurity
- JWT

### 데이터베이스
- MySQL + Docker(개발 환경)
- AWS RDS + MySQL(배포 환경)
- AWS S3 (이미지 저장)
